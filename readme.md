# MSRC CVE Lookup Script

This PowerShell script checks Microsoftâ€™s MSRC API for specific CVEs, retrieves detailed information from the CVRF update documents, and lists available remediations including related KB articles and affected products. Sometimes I have a CVE ID and I need more info, so I came up with this.

## Features

- Searches for one or more CVEs using Microsoftâ€™s official update API.
- Outputs remediation details: Knowledge Base (KB) numbers, descriptions, URLs, and affected product names.
- Warns if CVE data or product mapping is missing.

## Usage

1. Edit the script to specify the CVEs you want to search:

   ```powershell
   $CVEsWanted = @("CVE-2024-38202") # Add your CVEs here
   ```

2. Run the script in PowerShell.

## Requirements

- PowerShell 5.1 or later
- Internet access
